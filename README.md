# KairosRB


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kairos_rb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kairos_rb

## Usage

Check https://github.com/MatUrbanski/kairos_rb/blob/master/spec/kairos_rb/client_spec.rb for more details
